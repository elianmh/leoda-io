¿Cómo funciona?

Tres pasos. Una vez configurado, una sola conversación basta para producir.

1

Instala Cowork + el plugin

Bajas Claude Cowork (gratis) y arrastras el plugin de Leoda.io al chat. 2 minutos.

2

Conecta fal.ai

Te creas cuenta en fal.ai, agregas $5-10 de saldo, y corres /setup-leoda para guardar tu key.

3

Pide tu campaña

"Necesito un reel para mi café orgánico para Instagram" → te entrega imágenes, video, captions, calendario.
